Welcome to **Alexandro Bizeul, Anna Bordachard, and Santiago**'s LLM Streamlit project! 

Here you will find the different steps to follow in order to use and test our app! 


For iOS:

1) Import this project locally into one of your folders.

2) Make sure you are in the folder **cd path/to/LLM_Streamlit**.

3) Create a virtual environment **python3 -m venv .venv** and navigate to it: **source .venv/bin/activate**

4) Download the packages needed for the app: **pip install --upgrade pip** and **pip install -r requirements.txt**

5) Create a .env file **touch .env**

6) Fill in your .env file with the variable **OPENAI_API_KEY**=“sk-xxxx...” or edit directly from the terminal: **nano .env**

7) Run tests if you want: **python -m tests.test_file_name**

8) Launch the application! : **streamlit run app.py**


Pour Windows : 

1) Import this project locally into one of your folders.

2) Make sure you are in the folder **cd chemin\vers\LLM_Streamlit**.

3) Create a virtual environment **python -m venv .venv** and navigate to it: **.venv\Scripts\activate**

4) Download the packages needed for the app: **pip install --upgrade pip** and **pip install -r requirements.txt**

5) Create a .env file **New-Item .env**

6) Fill in your .env file with the variable **OPENAI_API_KEY**=“sk-xxxx...” or edit directly from the terminal: **notepad .env**

7) Run tests if you want: **python -m tests.test_file_name**

8) Launch the application! : **streamlit run app.py**
