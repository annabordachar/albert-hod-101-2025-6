### **Parsing Exercices**

For the package, there is a requirements.txt file that containe all used library

before testing create a venv or direct in your PATH : 

**pip install -r requirements.txt**



For the web crawler it need to be call on : 

pwd : **albert-hod-101-2025-6/Pasing_exercices**
command : **python Crawler/main.py**


For the email parser : 

pwd : **albert-hod-101-2025-6/Pasing_exercices**
command : **python Crawler/main.py**


### **Slack Bot Exercices**

All used library are basic we assume you have all required packages.

**Don't forget to set up your own tokens for the app and the "xapp-" and for the Bot token "xoxb-"**

To test the main.py on the "Slack_automation" it need to be call on : 

pwd : **albert-hod-101-2025-6/Slack_automation**
command : **python main.py**
